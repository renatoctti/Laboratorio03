Sistema de Moeda Estudantil

Sistema para estimular o reconhecimento do mérito estudantil através de uma moeda virtual.

Professores podem distribuir moedas aos alunos, que podem trocá-las por vantagens em empresas parceiras.

Como funciona o sistema
Cadastro

Alunos se cadastram no sistema informando seus dados pessoais e instituição de ensino.

Distribuição

Professores recebem 1000 moedas por semestre para distribuir aos alunos como reconhecimento.

Troca

Alunos podem trocar suas moedas por vantagens oferecidas pelas empresas parceiras.
